# MetaPhoto Backend

This project is a Node.js application created with Serverless framework

## prerequisites

- Node.js 16 or later
- npm or yarn

## installation:

1. Clone the repository

```console
git clone https://github.com/ldelacroixjc/sls_metaphoto.git
```
2. Navigate into the project directory.

```console
cd sls_metaphoto
```

3. Install the project dependencies: 

```console
npm install
```

## Running the Application:

1. Start the development server:

```console
sls offline
```
The application will be available at http://localhost:3000.


## Technologies used:

- Serverless (https://www.serverless.com/)
- AWS (https://aws.amazon.com/)
